# chatux
a tiny chat app built with React, Redux, SocketIO for learning purposes

# starting the project
for the server

`$ cd chatux/server`

`$ npm start`

for the client

`$ cd chatux/client`

`$ npm start`

you can find the project in `http://localhost:8080`
